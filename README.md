Palabraz Cheat
==============

This is a small script written in node.js to search
spanish words for the game palabraz.

### Usage
	node palabraz.js letters
Where letters is a string with all the letters given to you
left to right and up to bottom.
### Example
	A D F G
    H I J S
	P O L S
	R T A I

	node palabraz.js adfghijspolsrtai
